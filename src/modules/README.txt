Ce répertoire contient les modèles de squelettes utilisés pour :
* le site web
* les modèles de documents (reçu de don etc.)

Ne modifiez pas directement ces fichiers, modifiez-les depuis l'interface de Paheko, ou recopiez-les dans le répertoire data/files/skel.

Si vous modifiez des fichiers directement dans ce répertoire, ils seront écrasés lors de la prochaine mise à jour.
